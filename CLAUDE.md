# Angular App

Minimal Angular sandbox demonstrating component architecture and TypeScript class extension patterns.

## Tech Stack
- Angular
- TypeScript

## Project Structure
```
angular-app/
├── src/
│   └── app/
└── README.md
```

## Development
```bash
# Install dependencies
npm install

# Run development server
ng serve

# Build
ng build
```

## Key Notes
- This is a minimal sandbox/demo project, not a full application.
