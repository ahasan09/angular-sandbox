# Angular Sandbox

An Angular playground for experimenting with component overriding and extending TypeScript classes.

## Concepts Explored

- **Component overwrite** — overriding base component behaviour in Angular
- **Extended TypeScript classes** — inheritance patterns in Angular components

## Tech Stack

- Angular
- TypeScript

## Prerequisites

- [Node.js](https://nodejs.org/) v10+
- Angular CLI: `npm install -g @angular/cli`

## Getting Started

```bash
git clone https://github.com/ahasan09/angular-sandbox
cd angular-sandbox
npm install
ng serve
```

Open [http://localhost:4200](http://localhost:4200).
